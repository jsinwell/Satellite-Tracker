About This Project

This website is a personal project of mine I started nearly a year ago. I have always been fascinated with astronomy and space, and wanted to build something that could let me visually see every satellite that was in the sky right now. Moreover, as a computer science student, I also wanted to reinforce my software development skills. This project has taught me so many things, from the basics of HTML all the way to scraping a website using Node.js. I love every second I spend working on this, and I hope to continue to add features for the foreseeable future. I hope you enjoy what I've created.

Current Features:

  • Catalog of thousands of satellites (more planned)
  • Real-time data using Node.js and Axios
  • Search bar functionality
  • Fast forward or reverse time
  • Buttons to toggle skybox, atmospheric reflection
